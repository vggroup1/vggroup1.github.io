# Сайт VG Group (ВИДЖИГРУПП)

Статический сайт строительной компании. Без сборки и фреймворков — чистый HTML/CSS/JS.

## Как открыть
Откройте `index.html` двойным кликом в браузере. Всё работает локально.

## Как разместить в интернете
Загрузите содержимое папки `site/` на любой хостинг (или Netlify / GitHub Pages / Beget / Reg.ru и т.п.).
Стартовая страница — `index.html`.

## Структура
```
site/

├── js/layout.js            Шапка, меню, подвал, контакты (редактировать здесь)
├── js/main.js              Анимации, галерея, форма
└── assets/img/             Фотографии проектов
```

## Где менять контакты
Все контакты (телефон, e-mail, адрес, ссылки на ВК и MAX) заданы в одном месте —
в начале файла `js/layout.js`, объект `COMPANY`. Измените там — обновится на всех страницах.

## Форма заявки
Сейчас форма формирует письмо и открывает почтовый клиент с заполненными данными
(работает без сервера). Для приёма заявок прямо на сайт (без открытия почты)
подключите сервис форм (например, Formspree) или серверный обработчик —
правка в `js/main.js`, функция `initForm`.
